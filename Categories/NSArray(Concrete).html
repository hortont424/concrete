Parameters

block

The block to execute for each element in the array.

Return Value

An array including only elements for which the block returned YES.

Discussion

Executes a block for every element in the array, returning a list including only those elements for which block returned YES.

If block is nil, returns the original array.

reduce:

Executes a block for every element in the array, accumulating the results in the block’s first argument.

- (id)reduce:(id ( ^ ) ( id a , id b ))block

Parameters

block

The block to execute for each element in the array.

Return Value

The value returned by the final application.

Discussion

Executes a block for every element in the array, accumulating the results in the block’s first argument.

Each invocation merges two values into one, effectively eventually reducing the list to a single value.

If block is nil, returns nil.

zip:with:

Executes a block for corresponding pairs of elements in two arrays, returning a list of the results.

- (NSArray *)zip:(NSArray *)other with:(id ( ^ ) ( id a , id b ))block

Parameters

block

The block to execute for corresponding pairs of elements in the arrays.

Return Value

An array of the results of each application.

Discussion

Executes a block for corresponding pairs of elements in two arrays, returning a list of the results.

If block is nil, returns a list of NSNull.

If one array is longer than the other, the resultant array will be the length of the shorter array.
