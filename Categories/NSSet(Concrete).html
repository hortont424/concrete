partition:

Executes a block for every element in the set, partitioning the elements into two sets depending on the boolean return value of the block.

- (NSArray *)partition:(BOOL ( ^ ) ( id ))block

Parameters

block

The block to execute for each element in the set.

Return Value

An array with two elements: a set containing all elements for which the block returned NO, and all for which it returned YES, in that order.

Discussion

Executes a block for every element in the set, partitioning the elements into two sets depending on the boolean return value of the block.
